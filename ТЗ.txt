###########
Программа:
	1. Экран "Cписок уведомлений":
		а. Блок со списком уведомлений
			При клике на уведомление, открывается экран для редактирования уведомления(3.)
		б. Кнопка "Добавить уведомление"
			При клике по кнопке, открывается экран на котором можно создать уведомление(2.)
	2. Экран "Добавление уведомления":
		а. Поле для ввода заголовка уведомления
		б. Поле для ввода описание уведомления. 
		в. Кнопка "Добавить"
			При клике по кнопке, происходит добавление уведомления к существующему списку уведомлений
		г. Кнопка "Отменить"
			При клике по кнопке, происходит возврат на экран (1.)
	3. Экран "Редактирование уведомления":
		а. Поле для редактирования заголовка уведомления (с возможностью редактирования)
		б. Поле в котором отображается описание уведомления (с возможностью редактирования). 
		в. Кнопка "Сохранить"
			При клике по кнопке происходит сохранение уведомления и возврат на экран (1.)
		г. Кнопка "Удалить"
			При клике по кнопке происходит удаление уведомления из списка уведомлений и осуществляется возрат на экран (1.)
